# Cause → Recommended Fix Mapping

Generalized lookup: **Cause → Recommended Fix**

---

| Cause | Recommended Fix |
|---|---|
| Policy lapsed | Verify coverage and reinstate policy |
| Service before policy effective date | Bill to prior carrier or correct plan |
| Invalid member ID | Verify member ID and resubmit |
| Dependent aged out | Enroll dependent in own plan |
| Benefit exhausted | Wait for benefit reset or use alternate coverage |
| Waiting period not met | Reschedule after waiting period or appeal if exception applies |
| Excluded service | Appeal with medical necessity documentation |
| Coverage terminated | Enroll in COBRA or new plan; rebill if applicable |
| Multiple active policies | Determine COB and rebill in correct order |
| Pre-existing condition window | Reschedule after exclusion period or appeal |
| Out-of-territory care | Inform about travel rider; non-payable under domestic plan |
| Plan tier mismatch | Recommend benefit upgrade or alternate plan |
| Retroactive termination | Recover prior payments and rebill correct payer |
| Invalid or imprecise diagnosis code | Correct ICD-10 to full specificity and resubmit |
| CPT does not match diagnosis | Verify CPT and diagnosis alignment; resubmit |
| Unbundling violation | Bill global CPT, remove component codes |
| Upcoded E&M level | Downcode to documentation-supported level |
| Missing modifier | Append correct modifier and resubmit |
| Gender mismatch with procedure | Verify demographics and resubmit |
| Age-inappropriate procedure code | Use age-appropriate CPT |
| Mutually exclusive procedures | Remove conflicting code; bill primary only |
| Missing diagnosis code | Add ICD-10 diagnosis and resubmit |
| Frequency limit exceeded | Deny excess units; appeal with medical necessity |
| Invalid NDC | Verify NDC and resubmit |
| POS / CPT mismatch | Correct place of service or CPT |
| Wrong revenue code for facility | Apply correct revenue code per facility type |
| Out-of-network with no referral | Get PCP referral or use in-network provider |
| Expired provider license | Renew license; escalate to credentialing |
| Invalid or inactive NPI | Verify NPI in NPPES and resubmit |
| Provider not credentialed | Hold claims until credentialing is complete |
| Specialty mismatch | Verify rendering provider; reroute or correct |
| Wrong taxonomy code | Correct taxonomy on claim or enrollment |
| Sanctioned provider | Block payment; escalate to compliance |
| Missing provider signature | Sign and resubmit; switch to EDI |
| Facility not enrolled | Enroll facility or use in-network alternative |
| TIN/NPI mismatch | Submit corrected W-9 and resubmit |
| Missing supervising provider | Add supervising provider info and resubmit |
| Service address mismatch | Verify and correct service location |
| Missing prior authorization | Request authorization and resubmit |
| Expired prior authorization | Obtain new PA and resubmit |
| PA / billed CPT mismatch | Submit retro-PA for performed CPT |
| PA units exceeded | Pay to limit; request PA extension for excess |
| PA tied to wrong provider | Transfer or reissue PA to actual provider |
| PA tied to wrong facility | Submit PA for the correct facility |
| Missing referral | Obtain PCP referral and resubmit |
| Expired referral | Obtain new referral and resubmit |
| Lack of medical necessity | Appeal with additional clinical evidence |
| Late or missing admission notification | Submit late notification; appeal penalty if applicable |
| Step therapy not followed | Trial first-line therapy or submit exception |
| Experimental / investigational | File appeal or explore clinical trial pathway |
